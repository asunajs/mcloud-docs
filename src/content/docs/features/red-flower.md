---
title: 小红花领取
description: 直播间小红花领取功能（口令兑换）
---

小红花功能用于从移动云盘直播间领取小红花，通过口令兑换获取奖励。

## 基本用法

```javascript
import { drawRedFlower } from "./index.mjs";

await drawRedFlower(["云盘宠粉会员日", "会员日福利多多"]);
```

## 获取口令

口令主要从以下渠道获取：

1. **移动云盘 App 直播** — 直播间公告或评论区
2. **微信移动云盘视频号** — 视频号直播或动态
3. **小红书** — 搜索"移动云盘口令"，可能有人整理

## 多账号支持

配置多个账号时，会自动遍历领取：

```json
{
  "caiyun": [
    { "auth": "账号1认证信息", "nickname": "主号" },
    { "auth": "账号2认证信息", "nickname": "小号" }
  ]
}
```

## 注意事项

- 口令有时效性，建议获取后尽快使用
- 每个口令每天只能兑换一次
- 需要能访问移动云盘服务器
