title: 小红花领取
description: 直播间小红花领取功能（口令兑换）
---

小红花功能用于从移动云盘直播间领取小红花，通过口令兑换获取奖励。

## 基本用法

### 使用口令领取

```javascript
import { drawRedFlower } from "./index.mjs";

// 传入口令列表
await drawRedFlower(["云盘宠粉会员日", "会员日福利多多"]);
```

口令从小红书直播间获取，通常格式为：

```
口令1：云盘宠粉会员日
口令2：会员日福利多多
```

### 多个口令

```javascript
import { drawRedFlower } from "./index.mjs";

await drawRedFlower([
  "云盘宠粉会员日",
  "会员日福利多多",
  "签到领好礼",
]);
```

## 功能说明

### 自动执行流程

1. **获取用户信息** — 查询当前小红花数量
2. **埋点上报** — 模拟直播间访问行为
3. **首次参与奖** — 领取首次参与惊喜奖励
4. **口令兑换** — 逐个兑换口令获取小红花
5. **查询复活花** — 获取可用的复活花数量
6. **汇总结果** — 显示最终小红花数量

### 输出示例

```
==============
登录账号【138****1234】
当前红花：25
首次参与惊喜奖，获得 5 朵小红花
口令「云盘宠粉会员日」领取成功，获得 10 朵（云盘宠粉会员日奖品）
口令「会员日福利多多」领取成功，获得 5 朵（会员日福利多多奖品）
复活花：3
最终红花：45（+20）
```

## 多账号支持

如果配置了多个账号，会自动遍历所有账号领取：

```json
{
  "caiyun": [
    { "auth": "账号1认证信息", "nickname": "主号" },
    { "auth": "账号2认证信息", "nickname": "小号" }
  ]
}
```

执行时会依次为每个账号领取小红花。

## 获取口令

口令通常从小红书直播间获取，可以使用项目提供的脚本自动提取：

### 手动提取

1. 打开小红书，进入移动云盘直播间
2. 查看直播间公告或评论区的口令
3. 复制口令文本

### 自动提取脚本

项目提供了小红书口令提取脚本：

```bash
# 1. 先启动 Edge 浏览器（开启调试模式）
/Applications/Microsoft\ Edge.app/Contents/MacOS/Microsoft\ Edge --remote-debugging-port=9222

# 2. 在 Edge 中登录小红书

# 3. 运行脚本
npx tsx scripts/xhs-kouling.ts
```

脚本会自动：
- 连接 Edge 浏览器
- 打开用户主页
- 提取所有口令
- 格式化输出

## 自定义配置文件

如果需要使用非默认配置文件：

```javascript
import { drawRedFlower } from "./index.mjs";

await drawRedFlower(
  ["云盘宠粉会员日"],
  "/path/to/custom-config.json"
);
```

## 注意事项

1. **口令有效期** — 口令通常有时效性，建议获取后尽快使用
2. **每日限制** — 每个口令每天只能兑换一次
3. **网络要求** — 需要能访问移动云盘服务器
4. **账号状态** — 账号需要有效的认证信息
